Upcoming Courses

Build Better Websites with Chrome Developer Tools

This course will help you create bug-free, high-performing web pages and web apps using Google Chrome's developer tools. You'll learn how to inspect various aspects of a page, including the HTML structure, CSS styles, and JavaScript syntax and logic. In addition, you will learn useful tricks to evaluate page performance, including loading time, JavaScript execution, and content rendering.

Responsive Web Design Fundamentals

This course will feature a complete overhaul of our current responsive web design curriculum. You'll first learn how to think abstractly about the design decisions necessary to make content look great across multiple devices, then you'll move onto hands-on coding exercises that range from responsive typography and images to topics such as CSS layout with Flexbox and improving page performance.
