Upcoming Courses

3D Modeling for Virtual Reality Creation

In this course you will learn the basics of Autodesk's Maya software to create 3D generated imagery for use in Virtual Reality production. Concepts covered include an overview of Maya's interface, using the core modeling, animation, and rendering tools, and then exporting your project for use in various Virtual Reality environments.

Responsive Web Design Fundamentals

This course will feature a complete overhaul of our current responsive web design curriculum. You'll first learn how to think abstractly about the design decisions necessary to make content look great across multiple devices, then you'll move onto hands-on coding exercises that range from responsive typography and images to topics such as CSS layout with Flexbox and improving page performance.
